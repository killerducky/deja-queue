        DejaQueue

        DejaQueue needs a YouTube API Key to fetch information about the videos and playlists you import. You should also sign into your YouTube account in the embedded YouTube window (upper left).

        1. Go to the Google Cloud Console (link opens in your browser)
        2. Click the Create credentials dropdown.
        3. Select API key, keep the default options, and click Create
        4. Copy your new API key and paste it into the field below
